# Документация tg-bot-kbju

Telegram-бот для трекинга питания и подсчёта КБЖУ. Анализирует приёмы пищи по тексту или фото с помощью OpenAI, хранит данные в PostgreSQL, показывает статистику за день/неделю/4 недели.

**Версия:** 1.1.3
